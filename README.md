# 🛒 CodeMarket

O **CodeMarket** é um mercado fictício desenvolvido para praticar HTML, CSS e JavaScript — um marketplace de produtos de tecnologia (notebooks, periféricos, acessórios) voltado para programadores, estudantes e entusiastas de tecnologia.

🔗 **Site publicado:** https://arthurcodehub.github.io/mercado-ficticio/

## 🎯 Objetivo

Aplicar os conhecimentos adquiridos durante os estudos por meio da criação de um site completo, evoluindo o projeto gradualmente com novas funcionalidades e melhorias.

## ✨ Funcionalidades

- Header fixo com navbar horizontal no desktop e menu hambúrguer em telas menores
- Seção "Sobre" com layout em grid
- Catálogo de produtos em destaque e seção de ofertas com preço promocional
- Design responsivo, do celular a monitores ultrawide (breakpoints em 768px, 992px e 1400px)
- Identidade visual com inspiração em editor de código (cards estilo "janela de terminal", preços em fonte monoespaçada)

## 🚀 Tecnologias

- HTML5 (semântico, com atributos de acessibilidade — `aria-label`, `role`, etc.)
- CSS3 (variáveis/tokens de design, Grid, Flexbox, Media Queries)
- JavaScript (vanilla, sem frameworks)

## 🖥️ Como rodar localmente

Não precisa de instalação — é um site estático:

1. Clone o repositório: `git clone https://github.com/Arthurcodehub/mercado-ficticio.git`
2. Abra o arquivo `index.html` no navegador (ou use a extensão "Live Server" do VS Code)
